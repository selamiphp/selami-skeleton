### Installation

```bash
composer create-project selami/selami-skeleton myApp
```

### Testing locally

```bash
cd myApp
./selami server:run
```

### Available routes with the default installation:

* http://127.0.0.1:8080/
* http://127.0.0.1:8080/en/category/test-category-slug
* http://127.0.0.1:8080/en/2017/05/test-json-slug
